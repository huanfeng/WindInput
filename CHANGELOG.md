# 更新日志

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范。

## [0.1.0-alpha] - 开发中

### 新增

- 基于 Windows TSF 框架的输入法核心（C++ DLL）
- Go 输入引擎服务，支持拼音和五笔 86 输入
- 双拼输入支持（复用全拼词库与算法）
- 五笔拼音混输引擎
- 基于 Schema 方案的引擎驱动架构
- 候选窗口与工具栏 UI
- 中英文切换（Shift/Ctrl/CapsLock 可配置）
- 临时英文模式（Shift + 字母）
- 全角/半角、中英文标点切换
- 自动调频（词频学习）
- YAML 格式配置文件，支持热更新
- 基于 Wails + Vue 3 的图形化设置工具
- NSIS 安装包构建支持
- 码表过滤（通用规范汉字）

### 说明

> 这是清风输入法的首个公开版本，项目处于早期开发阶段。
> 功能和配置格式可能在后续版本中发生变化，升级时可能需要恢复默认配置。
