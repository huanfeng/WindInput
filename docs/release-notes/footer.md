## 安装

### 系统要求

- Windows 10 / 11（64 位）
- macOS 12 及以上（Apple Silicon / Intel，alpha）

### Windows 安装

**安装包（推荐）**

1. 下载下方 `WindInput-x.x.x-Setup.exe` 安装包
2. 双击运行安装程序，按提示完成安装
3. 安装完成后，按 `Win + Space` 或 `Ctrl + Shift` 切换到清风输入法即可使用

**便携版**

1. 下载下方 `WindInput-x.x.x-Portable.zip`
2. 解压到任意目录（如 `D:\Software\WindInput`）
3. 运行 `wind_portable.exe` 启动输入法

> 便携版数据保存在程序所在目录，适合 U 盘携带或多环境使用。

> **提示**：由于当前版本没有数字签名，安装时 Windows 安全中心（SmartScreen）可能会弹出拦截提示。点击「更多信息」→「仍要运行」即可继续安装。

### macOS 安装（alpha）

1. 下载下方 `WindInput-x.x.x-macOS.pkg`（universal，含输入法、后台服务、设置程序三件套）
2. 双击运行安装，按提示完成
3. 到「系统设置 → 键盘 → 文本输入 → 输入法」添加并切换到「清风输入法」（也可用 `⌃Space` 切换）

> 卸载：双击 `~/Applications/卸载清风输入法.app`。

> **提示**：macOS 版未做苹果公证，首次安装/启用可能需要在「系统设置 → 隐私与安全性」中点「仍要打开」放行；macOS 26 (Tahoe) 对未公证输入法限制更强。重新安装前建议先注销或重启，以清除系统输入法注册缓存。

## 注意事项

- 本项目目前处于 **alpha 阶段**，主要面向感兴趣的爱好者和早期测试用户
- 功能和配置文件格式可能在后续版本中发生变化
- 升级后如遇异常，请尝试删除配置目录 `%APPDATA%\WindInput\` 以恢复默认配置
