## 签署方式

### 首次贡献签署

当您首次向本项目提交 Pull Request 时，CLA Assistant 机器人会自动要求您签署本协议。您需要：

1. 在 PR 评论中回复：`I have read the CLA Document and I hereby sign the CLA`
2. 签署后，该 PR 及未来所有 PR 将自动通过 CLA 检查

### 签署记录

所有签署记录由 [CLA Assistant](https://cla-assistant.io/) 管理，可在项目仓库的 CLA 配置中查看。
